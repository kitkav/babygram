How to Run this website

1) have installed Apache and MySQL and replace all files to C:\xampp\htdocs\

2) In root folder open sql_import.php and edit $user = '' and $password = '' with your user.

3) In root folder open db.php and edit $user = '' and $pass = '' with your user.

4) And run localhost:80/sql_import.php into your browser to install the database.

5) Its all set to run the website. To run type localhost:80/index.php into your browser.
